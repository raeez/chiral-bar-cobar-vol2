**Volume II** of *Modular Homotopy Theory for Algebraic Factorization Algebras on Algebraic Curves*

## Research Frontiers

The remaining frontier problems are mathematical, not editorial. Each one has a precise theorem form and a named proof obligation.

1. **Curved-Dunn H^2 vanishing at g at least 2.** In `chapters/theory/curved_dunn_higher_genus.tex`, modular-bootstrap H^2 acyclicity plus `prop:modular-bootstrap-to-curved-dunn-bridge` imply `thm:curved-dunn-H2-vanishing-all-genera`. Jimbo-Miwa irregular-singular KZB supplies a separate boundary-composition input under boundary formal type and mixed Stokes hypotheses.
2. **DS-Hochschild bridge for class M.** In `chapters/theory/chiral_higher_deligne.tex`, `thm:chd-ds-hochschild` and `cor:universal-holography-class-M` give the weight-completed non-logarithmic statement. HPL transfer through the DS SDR, Arakawa C_2-cofiniteness, and HKR identification furnish the chain-level isomorphism `ChirHoch^bullet(W_k(g,f)) ~= H^bullet_DS(ChirHoch^bullet(V_k(g)))`. Logarithmic W(p) requires the Adamovic-Milas character-amplitude bound because Gurarie-Flohr logarithmic examples have unbounded correlation-function Massey growth.
3. **Periodic CDG for admissible KL at sl_N, N at least 3.** The mechanism runs through Arkhipov-Gaitsgory twisted Satake and Brylinski-Deligne reciprocity. The sl_2 case uses the Kubota class alpha_{S,u} = (1+(-1)^q)/2. Higher rank requires the determinant class in Pic(GR_G).
4. **Associator independence for chiral Deligne-Tamarkin.** The associator-dependent chain-level theorem is `thm:chd-deligne-tamarkin`. The associator-independent statement is controlled by the depth-3 Drinfeld-Kohno Massey class detected by Brown's motivic zeta(3); weights at least 4 require the corresponding grt_1 non-decomposability calculation.

## Mathematical Scope

The SC^{ch,top} heptagon is proved at chain level for five presentations after fixing a Drinfeld associator; the two derived-stack presentations are proved at the cohomological level. The E_3-topological structure is proved for affine Kac-Moody at non-critical level and for the verified finite-rung DS, freely generated PVA, and lattice lanes. All-rank W_N, Monster V^natural, and general conformal-VA cases require their finite-envelope, finite-orbifold BV, or chain-level topologization inputs. The E_infinity topological ladder is proved at finite spin depth once the independent translation and homotopy data are supplied. The W_infinity endpoint at generic lambda in Lambda_gen is controlled by all-spin GGL stabilization, inverse-limit Dunn compatibility, Yamada strong convergence, and truncation-compatible antighost homotopies. On the rational-lambda truncation locus Lambda_trunc, the ladder stabilizes at a finite E_{N+1}-topological rung.

## Build

All compiled output goes to `out/`.

```bash
make fast                    # quick converging build → out/main.pdf
make                         # full build: manuscript + working notes → out/
make release                 # full rebuild → out/ + iCloud
make test                    # compute test suite (pytest -q)
make verify-independence     # check ProvedHere tags vs @independent_verification decorators
make verify-independence-verbose   # same + full list of uncovered claims
make clean-builds            # remove /tmp/mkd-* isolated build directories
```

Each build runs in its own `/tmp/mkd-chiral-bar-cobar-vol2-<NS>/`
directory, so parallel builds use disjoint output trees. Set
`MKD_BUILD_NS` for warm rebuilds across invocations:

```bash
export MKD_BUILD_NS="build-$$"
make fast                         # warm on second call
```

Requires TeX Live 2024+ with pdflatex (memoir, EB Garamond, newtxmath). Python 3.10+ for compute suite.

## Independent Verification

Every theorem proved in this tree should have an independent verification path when a computation or disjoint derivation is available. The Python decorator in `compute/lib/independent_verification.py` enforces token-level disjointness between the programme-internal derivation and the external-source verification. See `notes/INDEPENDENT_VERIFICATION.md` for the rubric.

The key principle is mathematical: when a proof fails, repair the statement into its strongest true form. Scope is controlled by hypotheses, constructions, obstruction classes, and explicit computations, not by editorial labels.
