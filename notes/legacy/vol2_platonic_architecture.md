## 3. The form of each part

Every part of the platonic Vol II carries:

1. **Opening**: one sentence stating the question this part answers.
2. **Resolution**: the named theorem(s) and definition(s).
3. **Closing transition**: three sentences — what was established / next obstruction / construction that resolves it (per CG voice rule).

Every chapter carries:

1. **Title that names a mathematical object** (not a process / wave / round).
2. **Opening question** within the first ten lines, before the first definition.
3. **Definition with parenthetical first-principles meaning** for any standard
   concept (D-module, Ran, FM, Hodge bundle, $L_\infty$, Kuga-Satake, Humbert).
4. **One climactic theorem** named in the chapter title or first sentence.
5. **One worked example** that makes the theorem concrete (κ-tuple computed,
   $r(z)$-face exhibited, topologisation height verified).
6. **One closing transition** to the next chapter.

Every theorem carries:

1. **Statement in the lane in which its proof actually works** (chain-level or
   $(\infty,1)$-categorical; never a forced cross-lane translation).
2. **All licensing tags inline** (α/β/γ/δ/ε per v2 §3).
3. **Heptagon-face citation** when a face is invoked.
4. **`\ClaimStatusProvedHere` + IV decorator** for proved theorems;
   `\ClaimStatusConditional` + named hypotheses for conjectures;
   `\ClaimStatusEvidence` + numerical scope for finite-spin checks;
   `\ClaimStatusRetracted` for obsolete claims (pointing to Vol II's repair).
5. **Proof body** — not a sketch, not a citation-only black box.
6. **Cross-volume anchor** if the theorem is a Vol I / Vol III parallel.
