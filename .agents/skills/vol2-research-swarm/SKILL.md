# Vol II Research Swarm

This is the Codex counterpart of the Claude research-swarm workflow.

## Preconditions

- The user has explicitly authorized delegation, sub-agents, or parallel agent work.
- The task is frontier-scale, decomposable, and benefits materially from parallelism.

If either condition fails, fall back to `$vol2-frontier-research`.

## Frontier axes

Possible split axes:

- Swiss-cheese and operadic structure
- PVA descent and quantization
- bulk-boundary-line triangle
- 3D quantum gravity
- genuinely `E_1` examples and ordered-bar computations

## Swarm discipline

1. Identify the exact frontier target.
2. Decide which subtask stays on the critical path locally.
3. Split only genuinely independent sidecar tasks.
4. Assign disjoint ownership and write scopes.
5. Require every worker to respect existing edits and avoid revert behavior.
6. Integrate only after local reread and verification.

## Verification standard

- Every worker still owes hostile falsification.
- New formulas or compute claims still need multi-path verification.
- Cross-volume convention checks remain mandatory.

## Stop condition

- Do not call the swarm done because agents produced text.
- Integrate only after the resulting surface survives local reread, propagation, and the narrowest falsifying checks.
